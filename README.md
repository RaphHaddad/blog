# README

Raph's blog

## Local Development Setup

### 1. Clone the repository with the theme submodule

```cmd
git clone https://github.com/RaphHaddad/blog
git submodule init
git submodule update
```

### 2. Install Hugo

#### macOS

`brew install hugo`

#### Windows

`choco install hugo -confirm`

### 3. Run the blog

`hugo server -D`
